Configuration Instructions

To enable email sending, set these environment variables in your Replit Secrets:

Variable Description Example Required
EMAIL_ADDRESS Your email address for sending your.email@gmail.com Required
EMAIL_PASSWORD App password or email password your-app-password Required
SENDER_NAME Display name for sent emails Your Name Optional
SMTP_SERVER SMTP server (auto-detected if not set) smtp.gmail.com Optional
SMTP_PORT SMTP port (defaults to 587) 587 Optional
Email Provider Setup

  1. Enable 2-factor authentication on your Gmail account
  2. Go to Google Account settings → Security → App passwords
  3. Generate an app password for "Mail"
  4. Use your Gmail address for EMAIL_ADDRESS
  5. Use the generated app password for EMAIL_PASSWORD

Note: Regular Gmail passwords won't work - you need an app password.

  1. Sign in to your Microsoft account
  2. Go to Security settings → Advanced security options
  3. Under "App passwords," create a new app password
  4. Use your Outlook address for EMAIL_ADDRESS
  5. Use the generated app password for EMAIL_PASSWORD

SMTP settings are auto-detected for Outlook domains.

For other email providers, you'll need to manually set SMTP settings:

  • Find your provider's SMTP server and port
  • Set SMTP_SERVER and SMTP_PORT accordingly
  • Use your email address and password/app password
  • Ensure SMTP authentication is enabled in your email account
